# ひもパックバトル

Matter.js を使ったブラウザ向け 2D 物理アクションゲームです。プレイヤーパックに繋がった紐を引っ張り、敵パックを弾き飛ばしてスコアを伸ばします。

## ゲーム概要

- プレイヤーパックをドラッグ入力で加速させる
- 敵パックは一定以上の速度で弾き飛ばされると撃破
- ステージ上のアイテム取得で能力がレベルアップ
- GitHub Pages でそのまま公開できる静的構成

## 操作方法

- PC: プレイヤーパックの近くをマウスでドラッグ
- iPhone / スマホ: プレイヤーパックの近くを 1 本指でドラッグ
- リスタート: 右下の `リスタート` ボタン

## レベルアップ要素

- 紐の長さ Lv
- 紐の伸縮 Lv
- パックの大きさ Lv
- パックの重さ Lv

## ローカルでの起動方法

- `index.html` をブラウザで開く
- もしくは任意の静的サーバーで `002_himopack-battle/` を配信する

## GitHub Pages での公開方法

1. リポジトリを GitHub に push する
2. `Settings > Pages` を開く
3. `Deploy from a branch` を選ぶ
4. `main` / `/ (root)` を指定して保存する

## iPhone で遊ぶ際の注意点

- Pointer Events を使っているため Safari でも操作可能
- `touch-action: none` を設定して誤スクロールを抑制
- 横画面推奨だが、縦画面でも最低限表示可能
- 高 DPI 画面向けに `devicePixelRatio` を考慮
