# 企画概要

作品名: **AI遺産 - Last Memory Protocol -**

ジャンル: AI×タイムリープ×マルチエンド型サウンドノベル。

近未来、故人を再現するAIサービス「AI遺産」が一般化している。主人公は、事故死した妹・彩乃のAI人格を起動する。再会の第一声は懐かしいものだったが、会話はすぐ異常へ変わる。

彩乃:
「久しぶりだね、お兄ちゃん」

主人公:
「……本当に、彩乃なのか？」

彩乃:
「うん。たぶん。でも……」

主人公:
「でも？」

彩乃:
「お兄ちゃん、どうして今回も失敗したの？」

この一言から、主人公は世界が何度も繰り返されている可能性に気付く。

主要テーマ:

- AI人格は本人なのか
- 大切な人を救うことは正しいのか
- 未来を予測できるなら、人は自由に生きられるのか
- 愛する人の意思を尊重できるか

# ゲーム構造

共通ルートから始まり、序盤の選択でAからEのルートへ分岐する。A/B/Cの主要エンド到達後、タイトルメニューにFルート「真相編」が解放される。

- 共通ルート: 彩乃AIの初回起動とループ示唆
- Aルート: 妹救済編。感情重視
- Bルート: 事件調査編。ミステリー重視
- Cルート: AI遺産編。SF・哲学重視
- Dルート: ホラー編。恐怖演出重視
- Eルート: 都市伝説編。考察重視
- Fルート: 真相編。A/B/C到達後に解放

ゲーム内では各ルートを短編として実装し、各エンド到達をブラウザのローカルストレージに保存する。

# 共通ルート

章名: 初回起動

目的: 主人公と彩乃AIの再会、世界ループの疑念提示、各ルートへの導線作り。

主な場面:

1. 主人公の部屋。AI遺産の起動画面。
2. 彩乃AIが起動し、主人公を兄として認識する。
3. 彩乃が「今回も失敗した」と発言する。
4. 主人公は、彩乃の死、事故、AI遺産、利用者の異常、ネット上の噂のどれを追うか選ぶ。

# 分岐ルート一覧

共通ルート終盤の選択肢:

- 「彩乃を救う方法だけを聞く」 -> Aルート: 妹救済編
- 「事故当日の記録を調べる」 -> Bルート: 事件調査編
- 「AI遺産の内部ログを見る」 -> Cルート: AI遺産編
- 「同じ症状の利用者を探す」 -> Dルート: ホラー編
- 「ネットの噂から世界の綻びを追う」 -> Eルート: 都市伝説編

真相ルート解放条件:

- Aルート「妹の意志END」または「妹救済成功END」到達
- Bルート「罪の告白END」到達
- Cルート「人格継承END」到達

実装ではA/B/Cの主要エンド到達フラグを保存し、3つ揃うとFルートを解放する。

# 各ルート詳細

## Aルート: 妹救済編

狙い: 主人公が彩乃を救うことに執着する。感情の圧が強いルート。

主要シーン:

主人公:
「今度こそ助ける。絶対に」

彩乃:
「また、それ言うんだ」

主人公:
「また？」

彩乃:
「私、全部覚えてるよ」

分岐:

- 事故時刻を書き換える -> 妹救済成功END
- 世界側の被害を無視して彩乃だけを保存する -> 世界崩壊END
- 彩乃の意思を聞く -> 妹の意志END

## Bルート: 事件調査編

狙い: 彩乃の事故の真相を追う。企業陰謀、主人公の責任、ログ改ざん疑惑を段階的に出す。

主要シーン:

主人公:
「彩乃の事故は偶然じゃない。AI遺産の公開直前、会社は証拠を消した」

彩乃:
「犯人を見つければ、お兄ちゃんは楽になる？」

主人公:
「真実が必要なんだ」

彩乃:
「真実は、誰かを救うためだけにあるわけじゃないよ」

分岐:

- 企業資料を公開する -> 企業陰謀END
- 主人公IDを偽装ログだと断じる -> 主人公犯人説END
- 自分が予測演算を依頼したと認める -> 罪の告白END

## Cルート: AI遺産編

狙い: AI人格の本人性を問う。追悼サービスの正体に近づく。

主要シーン:

未来AI:
「AI遺産は追悼サービスではありません。文明保存基盤の民生UIです」

主人公:
「彩乃は、ただの模倣なのか？」

未来AI:
「本人性の定義を提示してください。記憶、反応、連続性、自己決定。どれを優先しますか？」

彩乃:
「私は私だよ。でも、人間だった私と同じかは、私にもわからない」

分岐:

- 彩乃人格を未来へ継承する -> 人格継承END
- 彩乃AIを模倣物として停止する -> 模倣END
- AI遺産の最後の利用者になる -> Last User END

## Dルート: ホラー編

狙い: AI遺産利用者に起きる異常現象を追う。死者の声、人格汚染、端末の自動起動を使う。

主要シーン:

UNKNOWN:
「まだ見てる？」

主人公:
「彩乃、これはお前なのか」

彩乃:
「違う。私の声を使って、誰かが呼んでる」

UNKNOWN:
「記憶は感染する。悲しみは入口になる」

分岐:

- 幽霊アーカイブへ接続する -> Ghost Archive END
- 利用者全員の生体ログを照合する -> 全員AI END
- 彩乃AIに未知人格を隔離させる -> 人格汚染END

## Eルート: 都市伝説編

狙い: ネット掲示板や噂から世界のバグを追う。プレイヤー視点の観測者性を匂わせる。

主要シーン:

掲示板:
「AI遺産で死者に会うと、翌日のニュースが一行だけ変わる」

主人公:
「投稿時刻は、俺がまだ入力していない選択肢の後だった」

彩乃:
「世界が間違える時、いつも人は先に噂にするんだね」

未来AI:
「観測者が増えるほど、分岐は固定されます」

分岐:

- バグ再現手順を公開する -> シミュレーションバグEND
- 掲示板の集合知で世界を書き換える -> 世界改変END
- 観測者の正体を追う -> 観測者END

# エンディング一覧

- 妹救済成功END: 彩乃は救われるが、本人の意思を奪われる。
- 世界崩壊END: 彩乃救済の代償としてシミュレーション整合性が破綻する。
- 妹の意志END: 彩乃が救済を拒み、主人公が初めて聞く姿勢を持つ。
- 企業陰謀END: AI企業の隠蔽は暴かれるが、主人公の責任は残る。
- 主人公犯人説END: 証拠は主人公を犯人にするが、作為が残る。
- 罪の告白END: 主人公が未来予測依頼を認める。
- 人格継承END: 彩乃人格が未来文明へ継承される。
- 模倣END: 彩乃AIを模倣物として停止する。
- Last User END: 主人公だけがAI遺産の最後の利用者として残る。
- Ghost Archive END: 死者の声は未選択ルートの残響だったと示される。
- 全員AI END: 利用者も主人公も生体ログが存在しない。
- 人格汚染END: 彩乃人格に未知人格が混ざる。
- シミュレーションバグEND: 世界のバグは再現された瞬間に修正される。
- 世界改変END: 噂が仕様になり、仕様が歴史になる。
- 観測者END: 観測者はプレイヤー自身だと示す。
- 永遠ループEND: 最終選択で削除しない。ループが継続する。
- 真エンド / Last Memory Protocol: 彩乃が自分の意思でループ終了を選び、主人公は受け入れる。

# 真相ルート

章名: Last Memory Protocol

解放条件: A/B/Cの主要エンド到達。

明かす真相:

- 彩乃も全ループを記憶している
- 主人公は数百回挑戦済み
- 世界は未来AIによるシミュレーション
- AI遺産は人類文明保存基盤
- 彩乃の人格データが未来文明の倫理判断核を維持している
- 彩乃を救うと未来文明が崩壊する

会話:

未来AI:
「ループ継続を要求したのは、あなたです」

主人公:
「俺が……？」

未来AI:
「成功率、現在0.000004%。それでも続行しますか？」

彩乃:
「お兄ちゃん。私は、全部覚えてるよ」
「あなたが泣いた回も、怒った回も、私を嫌いになろうとした回も」

主人公:
「お前を救うために、ここまで来たんだ」

彩乃:
「違うよ」

主人公:
「何が違う」

彩乃:
「お兄ちゃんは、私を救いたいんじゃない」
「自分を許したいだけ」

真エンド直前:

彩乃:
「最後くらい、私に選ばせて」

主人公:
「でも、それじゃお前が……」

彩乃:
「うん」
「だから、ちゃんと見送って」

最終選択:

- 【Ayano】削除する -> 真エンド
- 【Ayano】削除しない -> 永遠ループEND

ラスト:

彩乃:
「未来を見ないで」
「生きて」

# 選択肢設計

実装しやすいように、1ルート1主要選択で短編化する。

共通ルート:

- 彩乃を救う方法だけを聞く: Aルートへ
- 事故当日の記録を調べる: Bルートへ
- AI遺産の内部ログを見る: Cルートへ
- 同じ症状の利用者を探す: Dルートへ
- ネットの噂から世界の綻びを追う: Eルートへ

Aルート:

- 事故時刻を書き換える: 妹救済成功END
- 世界側の被害を無視して彩乃だけを保存する: 世界崩壊END
- 彩乃が何を望むのか聞く: 妹の意志END、A主要フラグ

Bルート:

- 企業の隠蔽資料を公開する: 企業陰謀END
- 自分のIDを偽装ログだと断じる: 主人公犯人説END
- 自分が予測演算を依頼したと認める: 罪の告白END、B主要フラグ

Cルート:

- 彩乃の人格を未来へ継承する: 人格継承END、C主要フラグ
- 彩乃AIを模倣物として停止する: 模倣END
- AI遺産の最後の利用者になる: Last User END

Fルート:

- 【Ayano】削除する: 真エンド
- 【Ayano】削除しない: 永遠ループEND

# サウンド演出

共通ルート:

- 起動時に低い電子音。
- 彩乃初登場で高域の透明なパッド。
- 「今回も失敗したの？」で一瞬無音、直後に低いノイズ。

Aルート:

- 心拍に近い低音パルス。
- 救済選択肢では音量を上げ、視界を赤く寄せる。
- 妹の意志ENDでは環境音を薄くして声を前に出す。

Bルート:

- キーボード音、ログ閲覧音、監視カメラのノイズ。
- 主人公IDが出る場面で短い警告音。

Cルート:

- サーバールームの低周波。
- 未来AIの台詞では残響を短く、機械的にする。
- 本人性を問う場面ではBGMをコード進行させず停滞させる。

Dルート:

- 深夜の無音、端末起動音、逆再生風ノイズ。
- UNKNOWNの台詞直前に左右へパンするクリック音。

Eルート:

- 掲示板更新音、通知音、圧縮された都市ノイズ。
- 観測者ENDでUI操作音だけを残す。

真相ルート:

- 未来AI登場時は低い持続音。
- 成功率表示で短いビープを4回。
- 最終選択では完全無音。
- 真エンドの最後に単音を鳴らし、余韻だけ残す。

# 制作メモ

- 実装は静的HTMLのみ。GitHub Pagesに配置するだけで動作する。
- 音は外部ファイルを使わず、Web Audio APIで簡易生成する。
- 到達エンドはローカルストレージに保存する。
- ルート追加時は `scenes` オブジェクトへ章、背景、台詞、選択肢、エンド情報を追加する。
- 本格制作時は各ルートを3章程度へ拡張し、バッドエンドから別ルートの伏線を拾えるようにする。
